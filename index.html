HannoGeoBot

The complete command reference for HannoGeo's Twitch bot.

!ahw [Google Maps link]
Submit a location to A Hanno World, our community map.
!pb
See what my current peak elo is on ranked duels.
!weather [city name]
Check the current weather. Default is Tallinn.
!time
See the current time in Tallinn right now.
!hug [name]
Give someone in chat a nice warm hug!
!slap [name]
Slap someone (or the bot when it's acting up).
!discord
Get the invite link to our awesome Discord server.
!iq [name]
Find out your (or someone else's) extremely accurate IQ.
!lurk
Let us know you're actively supporting by lurking.
!skip
If 3 people use this command, the currently playing song will be automatically skipped.
!pc
See my amazing PC specs powering the stream.
!setup
Check out the devices I use for my streaming setup.
!commands
See all the available commands. If you're here, you probably used this one already.

!gamble [amount of points]

!givepoints [username] [amount of points]

!leaderboard

!points [username]
